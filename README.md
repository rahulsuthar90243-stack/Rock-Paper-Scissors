<h2>Project Overview</h2>
<p>This is a simple yet well-structured <strong>Rock Paper Scissors</strong> game where:</p>

<ul>
      <li>The user plays against the computer</li>
        <li>Scores are tracked (User, Computer, Draws)</li>
        <li>Colored output is used for better terminal experience</li>
        <li>The project is built using modular programming principles</li>
    </ul>
